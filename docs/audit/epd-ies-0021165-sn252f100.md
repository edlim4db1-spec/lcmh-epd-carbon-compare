## Comparability notes
- SPECIAL FILE: the A1-A3 results table (page 32) is a rotated raster IMAGE embedded in the PDF, so its GWP values (fossil 143.73, biogenic 1.13E-01, luluc 1.77E-02, total 143.83) are not in the text layer and had to be read visually. The A4 table (page 34) IS native text (GWP-total 9.64).
- TRANSPOSED / per-stage layout: indicators are columns (GWP-fossil, GWP-biogenic, GWP-luluc, GWP-total, ...) and there is ONE product row ('Concrete produced at South Australia'). Column order on both pages: GWP-fossil, GWP-biogenic, GWP-luluc, GWP-total.
- Boundary table (page 18) declares A1-A3, A4, A5, C1, C2, C3, C4 and D as 'X' (in scope). HOWEVER the results section only publishes numeric tables for A1-A3 (page 32, Tables 8-11) and A4 (pages 34-35, Tables 13-16). NO results tables exist for A5, C1, C2, C3, C4 or D — those GWP cells are genuinely absent from the document and are left null (not_declared), never 0.
- Interpretation guidance (page 37) tells users to take the 'Cradle to Gate (A1-A3) GWP total' and add the 'A4' transport GWP — i.e. the EPD is effectively used as A1-A3 + A4 only, consistent with only those two stages having published results.
- Characterisation = EF 3.1 (page 29: 'aligned to and adopted from EN 15804+A2 version of Environmental Footprint 3.1'); GWP CFs use IPCC 2021 (AR6) 100-year baseline. Candidate's 'CML' referred only to the ADP-elements/fossil method (CML v4.8), not the overall set.
- GWP-total(A1-A3) = 143.83 kg CO2 eq per m3; GWP-total(A4) = 9.64 kg CO2 eq per m3. Combined cradle-to-gate + transport = ~153.5 kg CO2 eq/m3.
- Declared unit 1 m3 manufactured concrete; no density/mass conversion factor is provided in the EPD (mass_kg null).
- Mix SN252F100 = Futurecrete Normal Class, Type GL (EvoCem) cement + slag SCM, 25 MPa, produced in South Australia.
- Boundary (p18) claims A5, C1-C4 and D in scope, but the EPD publishes no results tables for them — shown as 'not reported', not zero.
